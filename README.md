# Basic Lab GNS3 — Enterprise Network Simulation

A basic enterprise network simulation built with **GNS3**, designed and configured as part of a networking lab project.

#Project Overview

This project simulates a company network infrastructure including device configuration, routing, and connectivity verification.

#Project Structure

```
├── Conf/                                  # Device configuration files
├── Project_Empresa.gns3                   # GNS3 project file
├── Proyecto_GNS3_Red_empresa.pdf          # Network design documentation
└── Imagenes_de_CONF_y_comprobaciones.pdf  # Configuration screenshots & verification
```

Tools Used

- [GNS3](https://www.gns3.com/) — Network simulation platform
- Cisco IOS routers and switches

How to Open

1. Install GNS3
2. Clone this repository
3. Open `Project_Empresa.gns3` in GNS3
4. Load device configurations from the `Conf/` folder

Documentation

See the included PDFs for full network diagrams, device configurations, and verification tests.

Made as part of ASIR (Network Systems Administration) coursework.
